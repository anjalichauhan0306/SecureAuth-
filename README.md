# SecureAuth 🔐

Production-level authentication system with:

- JWT Access + Refresh Token
- Role-Based Access Control (Admin/User)
- Password Hashing (bcrypt)
- Email Verification (Nodemailer)

## Tech Stack
- Node.js
- Express
- MongoDB

## Features
- Secure login/signup
- Token rotation
- Protected routes

## Setup
1. npm install
2. Create .env file
3. npm run dev
